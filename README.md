# Synced Grid Layout

By default, React Grid Layout does not come with responsiveness or sync features. The goal of this exercise is to implement a mechanism where the layout remains consistent across devices.

- **GitHub Repository**: [https://github.com/JainAadeesh/React--grid-layout]
- **Live Demo**: [https://react-grid-layout-iota.vercel.app/]

## How to Run

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Start Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```
